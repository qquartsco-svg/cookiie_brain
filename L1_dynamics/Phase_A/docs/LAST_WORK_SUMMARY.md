# 마지막 작업 요약

**규칙 (필수)**  
- 나(AI)만 작업하고 끝내면, 너(사용자)는 뭘 했는지 모른다.  
- 그래서 **작업한 뒤 반드시 마지막에** 다음을 보여준다:  
  1. **무슨 작업을 했는지**  
  2. **무슨 파일/문서를 수정·만들었는지 목록**  
  3. **경로를 적어서 누르면(또는 복사해서 Cmd+P에 붙이면) 바로 열 수 있게**  
→ 이 파일을 열면 "방금 한 작업"을 확인할 수 있다.
