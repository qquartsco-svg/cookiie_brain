# `_08_ice_age` — 빙하시대 진입 시뮬레이션

**L0_solar 서사 레이어 8번째 챕터**

`_07_polar_ice`의 최종 상태(T_pole, h_ice = 5m, 알베도 잠금)를 초기 조건으로 받아,
빙상-알베도 양의 피드백 루프가 수만 년에 걸쳐 LGM(최종 빙하기 최성기)으로
수렴하는 과정을 장기 시뮬레이션으로 검증한다.

## 알베도 피드백 구조

```
_07 출력: T_pole=−29°C, h_ice=5m, α_ice=0.55
        ↓
[_08 루프 시작]
극지 빙상 성장 → 알베도 상승 → 전지구 복사 강제 감소
        ↑                                      ↓
        └──────── 전지구 냉각 → 극지 온도 하강 ─┘
        (양의 피드백. 에어로졸 소산 후에도 루프 유지)
```

**에어로졸 소산(~10년) 후 루프 지속 이유:**
- `_07` 단계에서 알베도 잠금(α_ice=0.55) 달성
- 에어로졸 없어도 α_ice가 계속 태양 복사를 반사 → 냉각 유지
- 냉각이 빙상을 키우고, 빙상이 알베도를 높임 → 자기 강화

## 탈빙하기 임계 (Budyko Bistability)

LGM 상태(T = −0.5°C, SL = −122m)에서 현재 지구(T ≈ +14.2°C, SL ≈ 0m)로 복귀하기 위한
외부 복사 강제 임계값 탐색 결과:

| CO₂/화산 강제 (W/m²) | 20,000년 후 방향 |
|---|---|
| +19 | 빙하 성장 지속 (SL = −139 m) |
| +25 | 빙하 성장 지속 (SL = −191 m) |
| **+27~30** | **상전이 임계 — 빙하 소멸 / 폭증 분기점** |
| +30 | 빙하 완전 소멸 (SL = +124 m) |

- 임계 이하: 빙하-알베도 양의 피드백이 이겨 빙상 성장 지속
- 임계 이상: 피드백이 역방향 전환 → 빙상 붕괴
- 이것이 **Budyko EBM 양안정(Bistable) 상전이**의 수치 확인

### 시나리오별 분석

**S1 — 루시퍼 기준선 (검증 통과)**

LGM 실측값(SL −120m, 빙하선 ~60°N)을 **오차 2% 이내**로 재현.
루시퍼 충돌 + 창공 붕괴 시나리오가 실제 LGM을 설명한다는 수치적 근거가 된다.

**S2 — 에덴 대조군 (가장 중요한 결과)**

창공이 유지된 에덴 상태(T_global = +24°C)에서는 초기 빙상이 서서히 녹아
100,000년 후 43,000 km³의 소형 극지 빙관만 남는다.
해수면 변화 사실상 0.

> **핵심 결론**: 루시퍼 충돌이 없었다면 빙하시대는 발생하지 않는다.
> 창공(온실 캐노피)의 붕괴가 선행 조건이며,
> 충돌은 이미 불안정해진 시스템에 가해진 결정타다.

**S3 — 약한 충돌 (예상 외 결과)**

소행성급 충돌(T_pole = −15°C)이 오히려 S1보다 더 극단적인 빙하시대를 만든다.
SL = −196.6 m, 빙하선 52°N — S1보다 훨씬 심각하다.

물리적 원인:
- S1 T_pole = −30°C → B_acc = 0.30 m/yr (가우시안 최적점에서 멀어짐)
- S3 T_pole = −15°C → B_acc = **0.50 m/yr (최대값)** ← 최적 강설 온도와 일치
- 더 따뜻한 대기 = 더 많은 수분 = 더 많은 강설
- 빙상 성장이 65% 빨라지면서 알베도 피드백이 더 강하게 작동

> **통찰**: 빙하시대는 "가장 추운 상태"가 아니라
> "적당히 춥고 충분히 습한 상태"에서 가장 강력하게 발달한다.
> 극도로 차가운 공기는 건조해서 강설이 줄어드는 역설이 있다.

**S4 — 탈빙하기 실패 (심층 분석)**

+19 W/m²의 외부 강제에도 불구하고 빙상은 계속 성장한다.
T_global은 +14.6°C에서 시작하지만(현재 지구와 유사!) 점점 냉각.

실패 원인: 극지 온도가 −30°C에 고착되어 있어서
빙하 가장자리 기온이 −13°C → 융빙이 전혀 발생하지 않음.

탈빙하기 실제 임계: **+27~30 W/m²** 사이에 상전이 분기점 존재.

```
+19 W/m²: 빙하 성장 (SL −140m)  ─┐
+25 W/m²: 빙하 성장 (SL −191m)  ─┤ → 에너지 우물 탈출 실패
+27 W/m²: 임계 분기점             ─┤
+30 W/m²: 빙하 완전 소멸 (+124m)─┘ → 반대쪽 에너지 우물로 순간 이동
```

이 **양안정 상전이**는 현재 지구가 왜 수백만 년에 걸친
밀란코비치 주기 + 화산 CO₂ 복합 작용으로만 빙기에서 벗어날 수 있었는지를 설명한다.

### 물리 파이프라인

```
V_ice [km³]
    ↓ volume_to_geometry()  →  A_ice [km²], φ_ice [°N]
    ↓ global_albedo()       →  α_global
    ↓ radiative_forcing()   →  ΔQ [W/m²]
    ↓ global_temperature()  →  T_global [K]
    ↓ polar_temperature()   →  T_pole [K]  [PA=2.5]
    ↓ mass_balance()        →  B_net [m/yr]
    ↓ dV/dt = B_net×A×1e-3 →  V_ice (다음 스텝)
```

## 물리 모델 상세

### 1. 빙상 질량수지

**강설 누적 (Accumulation)**

$$B_{acc}(T_{pole}) = B_{max} \cdot \exp\!\left(-\frac{(T_{pole} - T_{opt})^2}{2\sigma^2}\right)$$

| 파라미터 | 값 | 의미 |
|---|---|---|
| $B_{max}$ | 0.50 m/yr | 최대 강설률 |
| $T_{opt}$ | −15°C | 최적 강설 기온 (수분↔결빙 균형) |
| $\sigma$ | 15 K | 가우시안 폭 |

**융빙 제거 (Ablation)**

$$B_{abl} = k_{abl} \cdot \max(0,\ T_{margin})$$

$$T_{margin}(\phi_{ice}) = T_{pole} + (T_{eq} - T_{pole}) \cdot \left(1 - \left(\frac{\phi_{ice}}{90°}\right)^{0.8}\right)$$

- $k_{abl} = 1.5\ m/yr/°C$ (PDD 계수)
- 빙하가 적도 쪽으로 내려올수록 가장자리 온도 상승 → 자연적 브레이크

### 2. 빙상 기하 (Volume → Geometry)

$$A = V / h_{mean}, \quad h_{mean} = 1.5\ km \quad \text{(LGM 보정)}$$

$$\sin\phi_{ice} = 1 - \frac{A}{2\pi R_\oplus^2}$$

### 3. 해수면 변화

$$\Delta SL = -\frac{(V - V_0) \cdot \rho_{ice}}{\rho_{water} \cdot A_{ocean}}$$

LGM 검증: $V = 5 \times 10^7\ km^3 \Rightarrow \Delta SL \approx -124\ m \approx -120\ m_{실측}$ ✓

### 4. 빙상-알베도 피드백

$$\alpha_{global} = \alpha_{ice} \cdot f_{ice} + \alpha_0 \cdot (1 - f_{ice}), \quad f_{ice} = 1 - \sin\phi_{ice}$$

$$\Delta Q = -\frac{S_0}{4} \cdot (\alpha - \alpha_0), \quad T_{global} = T_{ref} + S_{clim} \cdot (\Delta Q + Q_{CO_2})$$

| 상수 | 값 |
|---|---|
| $\alpha_{ice}$ | 0.65 (대륙 빙상) |
| $\alpha_0$ | 0.30 (기준 전지구) |
| $S_{clim}$ | 0.8 K/(W/m²) |
| $PA$ | 2.5 (극 증폭) |

### 5. 수치 안정성

Explicit Euler, dt = 50~100 yr. $\Delta t/\tau_{growth} = 50/5000 = 0.01$ — CFL 충분.
