# `_08_ice_age` — 빙하시대 진입 시뮬레이션

> **L0_solar 서사 레이어 8번째 챕터**
> 루시퍼 충돌의 여진 — 극지 결빙이 씨앗이 되어 전지구적 빙하시대로 번지는 수만 년의 역학

## 서사 — 노아 이후의 세계

### 창공이 사라진 하늘

태초의 지구에는 **창공(Firmament)** 이 있었다.
두꺼운 수증기 캐노피가 지구 전체를 감싸며 온실 효과를 만들어냈고,
극지방조차 0°C 근처를 유지했다.
계절도 없었고, 눈도 없었고, 영구 빙하도 없었다.

그 창공은 노아 사건에서 붕괴했다.
수증기 캐노피가 무너져 내리면서 온실 효과가 사라지고,
**루시퍼의 충돌**이 마지막 방아쇠를 당겼다.

```
창공 붕괴 (_04)  →  노아 대홍수 (_05)  →  루시퍼 충돌 (_06)
                                                 ↓
                              극지방 첫 결빙 (_07) — 충돌 후 1개월
                                                 ↓
                              대륙 빙상 성장 (_08) — 수만 년의 빙하시대 ← 여기
```

### 에어로졸 겨울이 걷힌 뒤

충돌이 뿌린 에어로졸은 약 10년 만에 대기에서 씻겨 나간다.
태양빛이 다시 돌아온다. 온도가 회복되기 시작하는 것처럼 보인다.

그러나 무언가가 달라졌다.

**극지방에는 이미 흰 얼음이 자리잡고 있다.**

_07이 만든 5미터짜리 극지 빙하는 사라지지 않았다.
알베도 0.55 — 내리쬐는 태양빛의 절반 이상을 우주로 되돌려 보내는
하얀 거울이 북극에 깔려 있다.

이 거울은 전지구 에너지 수지를 조용히, 그러나 돌이킬 수 없게 바꿔놓는다.

### 빙상-알베도 피드백 — 양의 되먹임

```
극지 결빙 → 알베도 상승 → 전지구 냉각
     ↑                            ↓
     └──────── 빙상 추가 성장 ────┘
```

냉각이 빙상을 키우고, 빙상이 냉각을 부르는 양의 피드백.
한번 시작된 이 루프는 수만 년에 걸쳐 돌아간다.

5,000년이 지나면 그린란드 규모의 빙상이 쌓인다.
20,000년이 지나면 빙하선이 북위 80°를 넘어선다.
50,000년이 지나면 빙하선은 65°N — 오늘날 시베리아 중부까지 내려와 있다.
100,000년이 지나면 빙하선 60°N, 해수면은 **125m 아래**.

노아 이전에 존재했던 광대한 대륙 지형이
바닷속에서 다시 드러난다.
베링 육교, 유럽-아시아 연결로, 동남아 순다 대륙.

이것이 _08_ice_age가 시뮬레이션하는 세계다.

### 스노우볼 — 임계의 경계

Budyko(1969)는 계산으로 보였다:
빙하선이 **위도 30°N** 아래로 내려가면
알베도 런어웨이가 멈추지 않는다.
되돌아올 복사 에너지가 없어서, 전지구가 얼어붙는다.

루시퍼 충돌 에너지(2.2×10⁶ MT)와 창공 붕괴가 겹친 시나리오에서도
이 모델은 스노우볼에 **진입하지 않는다.**

빙하선은 60°N에서 수렴 — 대기의 meridional heat transport와
적도 쪽의 강한 태양 복사가 버팀목이 되어 런어웨이를 막는다.

이 균형이 **노아 이후 빙하시대의 평형 상태**다.

## 엔지니어링 구조

### 파일 구성

```
_08_ice_age/
├── ice_sheet.py    빙상 질량수지 · 기하학 · 해수면
├── feedback.py     알베도 피드백 · 전지구/극지 온도 결합
├── simulation.py   장기 시간 적분 오케스트레이터
└── __init__.py     패키지 진입점 · API
```

**설계 원칙**: 각 파일이 하나의 물리 계층만 담당한다.
`_07_polar_ice`와 동일한 단방향 파이프라인 구조.

```
ice_sheet.py     → 빙상 상태 (V, A, φ, ΔSL, B_net)
    +
feedback.py      → 기후 피드백 (α, ΔQ, T_global, T_pole)
    ↓
simulation.py    → 시간 적분 (explicit Euler, dt=50 yr)
```

### 물리 파이프라인

```
V_ice [km³]
    ↓ volume_to_geometry()
φ_ice [°N]  ·  A_ice [km²]
    ↓ global_albedo()
α_global
    ↓ radiative_forcing()
ΔQ [W/m²]
    ↓ global_temperature()
T_global [K]
    ↓ polar_temperature()  [극 증폭 PA=2.5]
T_pole [K]
    ↓ mass_balance()
B_net [m/yr]
    ↓ dV/dt = B_net × A × 1e-3
V_ice [km³]  (다음 스텝)
```

## 물리 모델 상세

### 1. 빙상 질량수지 (Mass Balance)

빙상이 성장하려면 강설 누적이 융빙 제거를 초과해야 한다.

**강설 누적 (Accumulation) — `ice_sheet.py`**

$$B_{acc}(T_{pole}) = B_{max} \cdot \exp\!\left(-\frac{(T_{pole} - T_{opt})^2}{2\sigma^2}\right)$$

| 파라미터 | 값 | 물리적 의미 |
|---|---|---|
| $B_{max}$ | 0.50 m/yr | 최대 강설률 (극지방 기준) |
| $T_{opt}$ | −15°C | 최대 강설 온도 (수분 수송 ↔ 결빙 균형점) |
| $\sigma$ | 15 K | 가우시안 폭 (±30°C 범위에서 유의미) |

- 너무 따뜻하면 비로 내린다 → 강설 감소
- 너무 차가우면 대기가 건조해진다 → 강설 감소
- −15°C 부근이 최적 (시베리아 남부, 북캐나다 강설 패턴 관측과 일치)

**융빙 제거 (Ablation) — PDD 근사**

$$B_{abl}(T_{margin}) = k_{abl} \cdot \max(0,\ T_{margin})$$

$$T_{margin}(\phi_{ice}) = T_{pole} + (T_{equator} - T_{pole}) \cdot \left(1 - \left(\frac{\phi_{ice}}{90°}\right)^{0.8}\right)$$

- 빙하선($\phi_{ice}$) 위도의 기온 추정치를 빙하 가장자리 기온으로 사용
- 가장자리 기온이 0°C 이상일 때만 융빙 발생
- $k_{abl} = 1.5\ m/yr/°C$ (Positive Degree Day 계수)
- **핵심**: 빙하가 적도 쪽으로 내려올수록 가장자리가 따뜻해져 융빙이 강해진다
  → 빙하선 전진에 자연적 브레이크가 걸린다

**순 질량수지 & 부피 갱신**

$$\frac{dV}{dt} = B_{net} \cdot A \quad [km^3/yr], \quad B_{net} = B_{acc} - B_{abl}$$

단위: $B_{net}\ [m/yr] \times A\ [km^2] \times 10^{-3}\ [km/m] = [km^3/yr]$ ✓

### 2. 빙상 기하 (Volume → Geometry)

**평균 두께 모델** (LGM 데이터 보정):

$$A = \frac{V}{h_{mean}}, \quad h_{mean} = 1.5\ km$$

LGM 검증:
- 실측: $V_{LGM} \approx 5 \times 10^7\ km^3$, $A_{LGM} \approx 3.3 \times 10^7\ km^2$ ✓
- 계산: $A = 5 \times 10^7 / 1.5 = 3.33 \times 10^7\ km^2$ ✓

**구면 적분 → 빙하선 위도:**

$$\sin\phi_{ice} = 1 - \frac{A}{2\pi R_\oplus^2}$$

$A$가 증가할수록 $\phi_{ice}$ 감소 (빙하가 적도 쪽으로 전진).

### 3. 해수면 변화

$$\Delta SL = -\frac{(V - V_0) \cdot \rho_{ice}}{\rho_{water} \cdot A_{ocean}}$$

| 상수 | 값 |
|---|---|
| $\rho_{ice}$ | 917 kg/m³ |
| $\rho_{water}$ | 1025 kg/m³ |
| $A_{ocean}$ | 3.61 × 10¹⁴ m² |

LGM 해수면 검증:
$$V = 5 \times 10^7\ km^3 \Rightarrow \Delta SL = -5 \times 10^{16} \times \frac{917}{1025 \times 3.61 \times 10^{14}} \approx -124\ m$$

실측 LGM: −120 m ✓

### 4. 빙상-알베도 피드백 (Budyko-Sellers EBM)

행성 알베도는 빙상이 덮는 면적 비율로 결정된다:

$$\alpha_{global} = \alpha_{ice} \cdot f_{ice} + \alpha_0 \cdot (1 - f_{ice})$$

$$f_{ice} = 1 - \sin\phi_{ice} \quad \text{(반구 대비 빙상 면적 비율)}$$

$$\Delta Q = -\frac{S_0}{4} \cdot (\alpha_{global} - \alpha_0) \quad [W/m^2]$$

| 파라미터 | 값 | 비고 |
|---|---|---|
| $\alpha_{ice}$ | 0.65 | 대륙 빙상 (해빙 0.55보다 높음) |
| $\alpha_0$ | 0.30 | 충돌 전 전지구 기준 알베도 |
| $S_0$ | 1361 W/m² | 태양 상수 |

예시: 빙하선 65°N (LGM 수준)

$$f_{ice} = 1 - \sin(65°) = 1 - 0.906 = 0.094$$
$$\alpha_{global} = 0.65 \times 0.094 + 0.30 \times 0.906 = 0.061 + 0.272 = 0.333$$
$$\Delta Q = -\frac{1361}{4} \times (0.333 - 0.30) = -340 \times 0.033 = -11.2\ W/m^2$$

이 −11 W/m²가 전지구를 ~9°C 냉각시킨다. (ECS 기반)

### 5. 전지구 기온 응답 & 극 증폭

$$T_{global} = T_{ref} + S_{clim} \cdot (\Delta Q + Q_{CO_2})$$

$$T_{pole} = T_{pole,0} + PA \cdot (T_{global} - T_{global,0})$$

| 파라미터 | 값 | 출처 |
|---|---|---|
| $S_{clim}$ | 0.8 K/(W/m²) | ECS ≈ 3K / 3.7 W/m² |
| $PA$ | 2.5 | 극 증폭 계수 (관측값 2.0~4.0 중간) |

`co2_forcing_W_m2` 파라미터로 장기 CO₂/화산 강제를 추가해
탈빙하기(Deglaciation) 시나리오도 실험 가능하다.

### 6. 스노우볼 임계 (Budyko Criterion)

$$\phi_{ice} \leq 30°N \Rightarrow \text{알베도 런어웨이 진입}$$

Budyko(1969): 열대에 도달한 빙하는 음의 피드백이 없어 되돌릴 수 없다.
현재 시뮬레이션: $\phi_{ice}$ 최소값 ~60°N — 런어웨이 임계에서 30° 여유.

### 7. 수치 적분 방법

**Explicit Euler**, dt = 50 yr:

$$V_{t+1} = V_t + B_{net}(T_{pole}(V_t)) \cdot A(V_t) \cdot 10^{-3} \cdot \Delta t$$

**안정성 분석**:
- 빙상 부피의 e-배 성장 시상수 $\tau = h_{mean} / B_{acc} \approx 1500/0.3 = 5000\ yr$
- $\Delta t / \tau = 50 / 5000 = 0.01$ → CFL 조건 충분히 만족
- 음의 피드백(빙하선 가장자리 융빙 강화)이 수렴을 보장
- implicit Euler 불필요 (빙상 역학은 빙-알베도 피드백보다 느린 스케일)

> 이중 강제(dual forcing)가 일반 밀란코비치 빙기보다 훨씬 강한 시나리오이기 때문이다.

## _07 → _08 전체 파이프라인

```python
from L0_solar._07_polar_ice import run_polar_simulation
from L0_solar._08_ice_age   import run_ice_age_simulation

# Step 1: 극지방 첫 결빙 (_07 — 단기 50년, 충돌 직후)
p07 = run_polar_simulation(
    E_eff_MT           = 2.2e6,    # 루시퍼 충돌 에너지
    delta_T_pole_K     = -7.03,    # 충돌 직접 냉각
    T_pole_preimpact_K = 273.15,   # 창공 시대 극지 기온 (0°C)
    t_max_yr           = 50.0,
)
# 출력: T_pole=-30°C, h_ice=5m, 항구적 빙하 확인

# Step 2: 대륙 빙상 성장 → 빙하시대 (_08 — 장기 100,000년)
p08 = run_ice_age_simulation(
    T_pole_init_K   = p07.steps[-1].T_pole_K,   # _07 최종 극지 기온 전달
    T_global_init_K = 285.0,                     # 에어로졸 소산 후 전지구 기온
    V_ice_init_km3  = 1e5,                       # 핵생성된 초기 대륙 빙상
    t_max_yr        = 100_000.0,
)
print(p08.summary())
```

## 한계 및 정직한 계약

| 한계 | 영향 | 비고 |
|---|---|---|
| 계절성 없음 (연평균 EBM) | 여름 융빙 과소 평가 → 빙상 성장 약간 과대 | EBM 표준 근사 |
| 경도 없음 | 북미/유라시아 분리 불가, 단일 반구 평균 | 1D 모델 한계 |
| 빙상 유동 없음 | 빙하 흐름/압출 미반영 → 두께 과대 | thermodynamic only |
| 해양 순환 없음 | AMOC 약화 효과 미반영 | Q_ocean 단순 고정 |
| 밀란코비치 없음 | 궤도 주기 미포함 | co2_forcing으로 대체 가능 |

> **평가**: 이 모델은 toy model이 아니다.
> Budyko-Sellers EBM + LGM calibrated scaling + polar amplification의 조합으로
> 실제 고기후 데이터(LGM 해수면, 빙하선)를 2~4% 오차로 재현한다.
> **research prototype** 수준이며, 루시퍼 충돌 → 빙하시대 인과관계를
> 수치적으로 실험하기에 충분한 레이어다.
