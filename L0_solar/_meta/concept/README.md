# concept/ — 개념·서사·매핑 (문서만)

**역할**: 창조일·파이프라인 순서로 레이어를 **문서로** 볼 수 있게 함.  
**실제 코드**는 `solar/` 최상위의 `day1`, `day2`, …, `eden`, `underworld` 등에 있음.
