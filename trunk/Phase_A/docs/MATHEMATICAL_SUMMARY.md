## 📐 핵심 수식

### 1. 상태 공간

```
x ∈ ℝⁿ
```

### 2. 에너지

```
E(x) = -(1/2) x^T W x - b^T x
```

### 3. 보존력

```
F_conservative(x) = -∇E(x)
특성: ∇ × F_conservative = 0
```

### 4. 회전력 (Rotational Force)

```
F_rotational(x) = ω * J * (x - x_pole)
```

**여기서**:
- J: 반대칭 행렬 (skew-symmetric)
  ```
  J = [0  -1]  (2D)
      [1   0]
  ```
- ω: 회전 세기 (pole.strength * pole.rotation_direction)
- x_pole: 폴 위치 (회전 중심)

**특성**:
```
∇ × F_rotational ≠ 0  (회전 성분 존재)
```

### 5. 합성 힘

```
F(x) = -∇E(x) + ω * J * (x - x_pole)
```

## 🔄 동역학

### 1차 형식

```
ẋ = -∇E(x) + ω * J * (x - x_pole)
```

### 2차 형식 (현재 시스템)

```
ẍ = -∇E(x) + ω * J * ẋ
```

### 안정화된 자전

```
ẋ = -∇E(x) + ω * J * (x - x_pole) - γ * x
```

**조건**: `|ω| > γ` (자전 유지)

---

## 🎯 자전 발생 조건

### 필요 조건

```
∇ × F ≠ 0
```

**구현**:
```
F(x) = -∇E(x) + R(x)
여기서 ∇ × R(x) ≠ 0
```

**가장 간단한 형태**:
```
R(x) = ω * J * (x - x_pole)
```
