## 1️⃣ 좌표계와 상태 정의

### 현재 시스템

**상태 공간**:
```
x ∈ ℝⁿ
```

**에너지**:
```
E(x) = -(1/2) x^T W x - b^T x
```

**힘**:
```
F(x) = -∇E(x)
```

**현재 동역학**:
```
ẋ = -∇E(x)        (1차)
또는
ẍ = -∇E(x)        (2차)
```

---

## 2️⃣ 힘의 분해 (보존력 vs 회전력)

### 보존력 (Conservative Force)

**정의**:
```
F_conservative(x) = -∇E(x)
```

**특성**:
```
∇ × F_conservative = ∇ × (-∇E) = 0
```

**결과**: 회전 없음, 순환 운동 불가능

---

### 회전력 (Rotational Force)

**정의**:
```
F_rotational(x) = R(x)
```

**조건**:
```
∇ × R(x) ≠ 0
```

**결과**: 회전 성분 존재, 순환 운동 가능

---

### 합성 힘

**일반화**:
```
F(x) = -∇E(x) + R(x)
```

**특성**:
```
∇ × F = ∇ × (-∇E) + ∇ × R = ∇ × R ≠ 0
```

---

## 3️⃣ 자전이 자연 발생하는 조건

### 자전의 정의

**자전 = 상태가 폐곡선을 그리며 도는 것**

**수학적 표현**:
```
Limit cycle 존재
```

---

### 보존력만 있으면

**분석**:
```
∇ × (-∇E) = 0
→ 회전 없음
→ 절대 자전 안 나옴
```

**결론**: 보존력만으로는 자전 불가능

---

### 자전 발생 조건

**필요 조건**:
```
F(x) = -∇E(x) + R(x)
여기서 ∇ × R(x) ≠ 0
```

**이게 핵심**

---

## 4️⃣ 가장 간단한 자전 구조

### 2차원 회전 항

**가장 단순한 형태**:
```
R(x) = ω J x
```

**여기서 J는 반대칭(skew-symmetric) 행렬**:
```
J = [0  -1]
    [1   0]
```

**확장 (n차원)**:
```
J = 반대칭 행렬 (skew-symmetric)
```

---

### 동역학

**1차 형식**:
```
ẋ = -∇E(x) + ω J x
```

**2차 형식**:
```
ẍ = -∇E(x) + ω J ẋ
```

**특징**:
- ω ≠ 0 이면 자전 성분 발생
- ω = 0 이면 현재 시스템

---

## 5️⃣ 자전 안정 조건

### 순환만 있으면

**문제**: 발산하거나 붕괴

**해결**: 감쇠 항 추가

---

### 안정화된 자전

**수식**:
```
ẋ = -∇E(x) + ω J x - γ x
```

**여기서**:
- γ > 0: 감쇠
- ω: 회전 세기

**자전이 유지되려면**:
```
|ω| > γ
```

**또는 에너지 주입 항이 있어야 함**

---

## 6️⃣ 위성(달) 필요 조건 분석

### 물리적 사실

**행성은 달 없이도 자전한다**

**이유**:
- 자전은 각운동량 때문
- 위성 때문이 아님

**예시**:
- 수성, 금성, 목성, 토성
- 위성 없어도 자전

---

### 수학적 분석

**자전 발생 조건**:
```
∇ × F ≠ 0
```

**위성이 필요한 경우**:
- 세차 안정화
- 주기 동기화
- 다중 주파수 결합

**결론**:
- ❌ 자전에는 필요 없음
- ✅ 복잡 리듬에는 도움됨

---

### 달의 실제 역할

**조석력 (Tidal Force)**:
```
F_tidal ∝ GM_moon / r^3
```

**효과**:
- 자전 속도 조절
- 조석 마찰로 에너지 소산
- 자전축 안정화
- 대류 구조 강화

**수학적 표현**:
```
F_total = -∇E(x) + R(x) + F_tidal(x, t)
```

**결론**: 달은 자전의 원인이 아니라 조율자

---

## 7️⃣ 공전은 무엇인가?

### 공전의 정의

**공전 = 2개 이상의 질량 중심이 있을 때 생김**

---

### 수학적 표현

**두 상태 벡터**:
```
x₁, x₂ 존재
```

**힘**:
```
F₁ = -∇E₁(x₁) + G(x₁ - x₂)
F₂ = -∇E₂(x₂) + G(x₂ - x₁)
```

**여기서 G는 상호 결합 항**

---

### 공전 조건

1. **인력** ∝ 1/r
2. **관성 존재**
3. **에너지 보존 또는 준보존**

---

## 8️⃣ 결론 정리

### 자전 조건

- ✅ 회전 성분 존재 (∇ × R ≠ 0)
- ✅ 또는 limit cycle 존재
- ❌ 위성 필요 없음

---

### 세차 조건

- ✅ 다차원 비대칭 곡률
- ✅ 복합 주파수
- ✅ 결합 항 존재

---

### 공전 조건

- ✅ 최소 2개 이상의 질량 중심
- ✅ 상호 결합 포텐셜
- ✅ 관성 포함

---
