# 자전과 공전: 정적 포텐셜에서 동적 운동으로의 전환

**작성일**: 2026-02-21  
**버전**: 0.1.3

---

## 🌑 현재 상태: 정적 포텐셜 덩어리

### 현재 Cookiie 코어의 특성

**수학적 구조**:
```
상태: x (위치만)
업데이트: x' = -∇E(x)
에너지: E(x) = -(1/2) * x^T W x - b^T x
```

**물리적 비유**:
- 중력장 위에 놓인 정지 질량
- 최소점(우물)으로 흘러가서 멈춤
- 한 번 도착하면 끝 (고정점 attractor)

**특징**:
- ✅ 안정성 있음 (에너지 최소점으로 수렴)
- ✅ 선호 있음 (우물 = 기억)
- ✅ 기억 씨앗 있음 (W, b)
- ❌ 시간 구조(리듬) 없음
- ❌ 위상(phase) 없음
- ❌ 주기성 없음

**문제점**:
- 완전히 정지한 최소점은 정보도, 판단도, 기능도 하지 않음
- 그냥 "안정"일 뿐

---

## 🌪 왜 "자전"이 필요한가?

### 기능을 위한 조건

뇌든, 로봇이든, 기능하려면:
- **상태 A** ↔ **상태 B** 반복적 오가기 필요
- 구분이 생기고
- 타이밍이 생기고
- 정보 처리가 가능해짐

**비유: 낮/밤**:
- 낮 = 상태 A (활성)
- 밤 = 상태 B (휴식)
- 이 반복이 있어야 생명체처럼 기능함

**현재 시스템**:
- 완전 감쇠(overdamped)
- 외부 에너지 주입 없음
- 진동 없음
- → **자전(내부 리듬)이 없음**

---

## 🌟 자전이란 무엇인가? (수학적 정의)

### 자전 = 자율 진동(Oscillation)

**핵심**: 내부 상태가 스스로 주기적으로 변하는 메커니즘

### 구현 방법 (4가지)

#### ① 관성 추가 (2차 미분)

**현재 (1차)**:
```
x' = -∇E(x)
```

**관성 추가 (2차)**:
```
x'' + γx' + ∇E(x) = 0
```

**효과**:
- 최소점 주변을 지나침
- 반대쪽으로 튐
- 다시 돌아옴
- → **진동 발생**

#### ② 리미트 사이클 (비선형 진동자)

**구조**: Hopf bifurcation
- 최소점이 아니라 고정 궤도 생성
- 점이 아니라 궤도가 안정 구조

#### ③ 약한 불안정성 + 복원력

- 완전 최소점 대신 주변을 맴돎
- 미세한 진동 유지

#### ④ 에너지 주입 + 소산 균형

- 외부 input 또는 내부 발진기
- 살아있는 시스템처럼 유지되는 진동

---

## 🌍 세차운동 (Precession)

### 세차운동이란?

**단순 진동과의 차이**:
- 단순 진동: 한 방향으로 왕복
- 세차운동: 위상 공간에서 원형/나선 궤도

**필요 조건**:
- 완전 대칭이 아닌 필드
- 비선형성
- 다차원 곡률 차이

**결과**:
- 한 방향으로만 단순 왕복이 아니라
- 위상 공간에서 복잡한 궤도 생성
- → **"자전"에 해당**

---

## 🔄 공전이란 무엇인가?

### 공전의 정의

**자전**: 내부 리듬 (단일 attractor 주변의 진동)

**공전**: 
- 서로 다른 attractor들 사이의 천천한 이동
- 또는 고차 상태 공간에서의 주기적 궤도

**관계**:
- **자전이 먼저다**
- 내부 리듬이 있어야 외부와의 상호작용이 생김

---

## 🎯 포텐셜 필드에서 운동이 생기려면

### 개념적으로 필요한 것 (4가지 중 하나 이상)

#### ① 관성 (Inertia)

**의미**: 에너지를 바로 소모하지 않고 "보유"한다

**수학적 표현**:
- 상태에 "속도" 포함
- 1차 미분 → 2차 미분

**현재 시스템**:
- `state_vector = [x, v]` (이미 포함되어 있음!)
- 하지만 완전 감쇠로 진동 없음

#### ② 비보존력 (Non-conservative Force)

**의미**: 순환 성분이 생긴다

**수학적 표현**:
- ∇E 말고 회전 성분 추가
- `curl(g) ≠ 0` (비퍼텐셜 성분)

#### ③ 에너지 주입 (Energy Injection)

**의미**: 외부 input 또는 내부 발진기

**예시**:
- 주기적 외부 자극
- 내부 발진기 (oscillator)

#### ④ 최소점 대신 Limit Cycle

**의미**: 점이 아니라 궤도가 안정 구조

**구조**: Hopf bifurcation

---

## 🧠 뇌적 해석

### 현재 구조

- **기억** = 에너지 우물
- **판단** = 최소점 선택

**하지만 아직**:
- ❌ 리듬 없음
- ❌ 시간 구조 없음
- ❌ 위상 없음

### 운동이 생기면

- ✅ 위상(phase)이 생김
- ✅ 타이밍이 생김
- ✅ 주기성이 생김
- → **그때부터 "기능"처럼 느껴진다**

## 🌑 비유로 다시 정리

### 현재 상태

**필드 위에 고여 있는 중력 덩어리**

### 관성이 들어가면

**회전이 시작됨 (자전)**

### 비대칭이 들어가면

**세차운동**

### 다중 attractor가 경쟁하면

**공전**

---

## 🔥 핵심 전환 포인트

### 정적 시스템 → 동적 시스템

**이건 단순 개선이 아니라 시스템의 차원 상승이다**

**현재**: 1차 시스템 (위치만)
**목표**: 2차 이상 시스템 (위치 + 속도 + 위상)

---

## ⚠️ 중요한 설계 분기점

### 질문: 어떤 종류의 운동을 원하는가?

#### 옵션 1: 안정성 유지하면서 미세 진동

**특징**:
- 최소점 근처에서 작은 진동
- 안정성 유지
- 약한 관성 + 감쇠

**수학적 표현**:
```
x'' + γx' + ∇E(x) = 0  (γ > 0, 약한 감쇠)
```

**비유**: 시계추처럼 규칙적 진동

#### 옵션 2: 진짜 살아있는 Chaotic 리듬

**특징**:
- 복잡한 비선형 진동
- 위상 공간에서 복잡한 궤도
- 생체 리듬(세타/감마)처럼

**수학적 표현**:
- Hopf bifurcation
- Limit cycle
- Chaotic attractor

**비유**: 뇌파처럼 복잡한 리듬

## 📊 현재 시스템 분석

### 이미 있는 것

1. **2차 동역학 구조**: `state_vector = [x, v]` (위치 + 속도)
2. **관성 개념**: 속도 업데이트 포함
3. **에너지 계산**: 운동 에너지 + 퍼텐셜 에너지

### 없는 것

1. **진동 메커니즘**: 완전 감쇠로 진동 없음
2. **위상 구조**: 주기성 없음
3. **에너지 주입**: 외부 자극 없음
4. **비선형 진동**: 단순 수렴만 있음
