# L4_analysis — Layer 4 : 분석 모듈

CookiieBrain의 **읽기 전용 측정 레이어.**  
`L0_solar`·`L1_dynamics`·`L3_memory`가 만들어낸 결과를 측정하고 분석한다.  
**이 레이어는 시스템을 절대 변경하지 않는다.** 관찰만 한다.

---

## 이 레이어가 하는 일

- 궤적·물리량을 통계역학·장론·기하학으로 분석
- 6개의 분석 계층(Layer_1~6)이 순서대로 쌓임
- 각 계층은 이전 계층 위에서 더 깊은 분석 수행

---

## 폴더 구조

```
L4_analysis/
├── README.md                ← 지금 읽고 있는 문서
├── brain_analyzer.py        ← 전체 분석 오케스트레이터
├── ocean_simulator.py       ← 해양 시뮬레이션 분석
├── Layer_1/                 ← 통계역학 정식화
├── Layer_2/                 ← 다체·장론
├── Layer_3/                 ← 게이지·기하학
├── Layer_4/                 ← 비평형 일 정리
├── Layer_5/                 ← 확률역학
└── Layer_6/                 ← 정보 기하학
```

---

## 분석 계층별 역할

| 계층 | 이름 | 핵심 파일 | 하는 일 |
|------|------|-----------|---------|
| Layer_1 | 통계역학 | `statistical_mechanics.py` | 궤적을 확률·열역학 언어로 번역. Langevin+FDT 기반 |
| Layer_2 | 다체·장론 | `nbody.py` | N체 상호작용·장론으로 분석 확장 |
| Layer_3 | 게이지·기하학 | `gauge.py` | 코리올리 게이지를 위치 의존 반대칭 연산자로 확장 |
| Layer_4 | 비평형 일 정리 | `fluctuation_theorems.py` | 비평형 과정에서 자유 에너지 추출 (Jarzynski 등) |
| Layer_5 | 확률역학 | `stochastic_mechanics.py` | 궤적 → 밀도 ρ(x,t) 진화 관점으로 전환 |
| Layer_6 | 정보 기하학 | `geometric_phase.py` | Fisher 정보 계량으로 매개변수 공간 기하 분석 |

---

## 계층 흐름

```
Layer_1 (통계역학)
  → Layer_2 (다체·장론)      : 평형 → 다체 상호작용
    → Layer_3 (게이지·기하학) : 전역 게이지 → 위치 의존
      → Layer_4 (비평형)      : 평형 열역학 → 비평형 등식
        → Layer_5 (확률역학)  : 궤적 → 밀도 진화
          → Layer_6 (정보기하): 확률 → 매개변수 기하학
```
