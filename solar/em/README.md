  — 세차운동. 아무도 보지 못하는 어둠 속에서.

  B ∝ 1/r³. 가까이에서만 강하고, 멀어지면 급격히 약해지는 방어막.

  그림자가 없었다.

## 빛이 있으라

```
  빛이 있으라 하시니 빛이 있었고

  And God said, "Let there be light,"
  and there was light.

  — Genesis 1:3
```

**2026-02-25, v1.3.0.**

태양의 질량(1.0 M☉)으로부터 광도가 유도되었다.

```
  L = M^α = 1.0^4.0 = 1.0 L☉
```

빛은 역제곱 법칙을 따라 공간으로 퍼져나갔다.

```
  F(r) = L / (4πr²)
```

그 순간, 어둠 속의 질량들에 형태가 생겼다.

| 행성 | 거리 (AU) | 조도 (W/m²) | 평형 온도 (K) |
|------|:---------:|:-----------:|:------------:|
| 수성 | 0.387 | 9,083 | 437 |
| 금성 | 0.723 | 2,601 | 229 |
| **지구** | **1.000** | **1,361** | **254** |
| 화성 | 1.524 | 586 | 210 |
| 목성 | 5.203 | 50 | 103 |
| 토성 | 9.555 | 15 | 81 |
| 천왕성 | 19.218 | 3.7 | 58 |
| 해왕성 | 30.110 | 1.5 | 47 |

수성은 9,083 W/m²로 타올랐다.
해왕성은 1.5 W/m²의 희미한 빛을 받았다.
그 사이, 지구는 1,361 W/m²의 빛 아래에서 254 K의 온도를 갖게 되었다.

낮과 밤이 나뉘었다.
그림자가 생겼다.
존재의 의미가 시작되었다.

## 이 폴더가 담고 있는 것

이 폴더(`solar/em/`)는 중력장 위에 얹혀진 **환경 필드 레이어**다.
core/(물리 엔진)가 만든 궤도와 자전축을 읽기만 하고, 수정하지 않는다.
관측자 모드. 빛은 존재를 드러내지만, 궤도를 바꾸지 않는다.

### 구조

```
solar/em/
│
├── solar_luminosity.py    Phase 5: 빛이 있으라
│   │                      L = M^α → F = L/(4πr²) → T_eq → P_rad = F/c
│   │                      광도에서 조도, 온도, 복사압까지
│   │                      태양의 빛과 열. 존재에 형태를 부여하는 레이어.
│   │
│   │                      복사(photons)와 플라즈마(solar wind)는 독립.
│   │                      복사압은 여기서 유도한다. solar_wind에서가 아니다.
│   │
├── magnetic_dipole.py     Phase 2: 자기쌍극자장
│   │                      B(r) ∝ 1/r³
│   │                      자전축에서 11.5° 기울어진 자기축.
│   │                      세차운동과 동기 — 축이 돌면 자기장도 돈다.
│   │                      지구의 보이지 않는 갑옷.
│   │
├── solar_wind.py          Phase 3: 태양풍 (플라즈마)
│   │                      P_sw ∝ 1/r², Φ ∝ 1/r², IMF ∝ 1/r²
│   │                      태양에서 뿜어져 나오는 하전 입자.
│   │                      복사(photon)가 아닌 플라즈마(plasma)만 다룬다.
│   │
├── magnetosphere.py       Phase 4: 자기권
│   │                      B² vs P_sw → 마그네토포즈 경계
│   │                      r_mp ≈ 7.6 R_E (지구)
│   │                      차폐율 0.78. 극 쿠스프 침투. 자기꼬리.
│   │                      "밀어내는 힘(자기장) vs 밀려오는 힘(태양풍)"의 균형.
│   │
├── _constants.py          EPS_ZERO, EPS_GEOM 중앙 관리
│
└── README.md              이 문서. 빛이 있으라.
```

### 의존 규칙

```
core/* 는 em/* 를 절대 import하지 않는다.
em/* 는 core/ 의 Body 상태를 읽기만 한다 (pos, vel, spin_axis).
em/* 과 cognitive/* 는 서로 참조하지 않는다.
"통합"은 demo 또는 coupling 레이어에서만 한다.
```

### 물리 분리 원칙

```
복사 (photons)  →  solar_luminosity.py  →  L → F → P_rad = F/c
                                                    독립
플라즈마 (wind)  →  solar_wind.py       →  P_sw = ρv²
                                                    독립
자기장 (dipole)  →  magnetic_dipole.py  →  B ∝ 1/r³
                         │
                         ▼
자기권 (shield)  →  magnetosphere.py    →  B² vs P_sw  (복사 미참여)
```

복사와 플라즈마는 같은 단위(Pa)를 가지지만 물리적 원인이 다르다.
복사압(P_rad ≈ 4.54 μPa)은 태양풍 동압(P_sw ≈ 2 nPa)보다 ~2,270배 크지만,
광자는 자기장과 상호작용하지 않으므로 자기권 경계는 P_sw만으로 결정된다.
이것이 두 모듈을 분리한 물리적 근거다.

## 지배 방정식

| 물리량 | 수식 | 모듈 |
|--------|------|------|
| 광도 | L/L☉ = (M/M☉)^α | solar_luminosity |
| 복사 조도 | F(r) = L/(4πr²) | solar_luminosity |
| 복사압 | P_rad = F/c | solar_luminosity |
| 평형 온도 | T_eq = [F(1-A)/(f·ε·σ)]^¼ | solar_luminosity |
| 자기쌍극자 | B(r) = B₀(R/r)³[3(m̂·r̂)r̂ - m̂] | magnetic_dipole |
| 태양풍 동압 | P_sw(r) = P₀(r₀/r)² | solar_wind |
| 입자 플럭스 | Φ(r) = Φ₀(r₀/r)² | solar_wind |
| 행성간 자기장 | B_sw(r) = B₀_sw(r₀/r)² | solar_wind |
| 마그네토포즈 | r_mp = R(k·B₀²/P_sw)^(1/6) | magnetosphere |

## 빛이 있은 후

```
  빛이 있기 전에도 우주는 존재했다.
  질량이 공간을 휘고, 중력이 궤도를 만들고,
  자기장이 방패를 세우고, 태양풍이 밀려왔다.

  그러나 빛이 없으면 — 형태가 없다.
  형태가 없으면 — 그림자가 없다.
  그림자가 없으면 — 경계가 없다.
  경계가 없으면 — 존재의 의미가 없다.

  빛이 켜진 순간,
  수성은 타올랐고, 해왕성은 희미하게 빛났고,
  지구는 254 K의 온도 위에서
  자기장 방어막 안쪽에서, 태양풍을 걸러내며,
  빛을 받기 시작했다.

  이것이 환경의 시작이다.
  이것이 존재의 시작이다.

  빛이 있으라 하시니 — 빛이 있었고,
  빛이 있으매 — 모든 것이 의미를 갖기 시작했다.
```
