# 빛이 있으라 / Let There Be Light

**v1.3.0 · 2026-02-25**

---

## 빛이 있기 전

이 공간에는 이미 모든 것이 있었다.

질량이 있었다. 중력이 있었다.
태양은 1.0 M☉의 질량으로 공간을 휘었고,
8개의 행성은 보이지 않는 중력선을 따라 어둠 속에서 궤도를 돌고 있었다.

지구는 23.44°로 기울어진 채 자전했고,
그 기울어진 축은 태양과 달의 토크를 받아
25,000년 주기로 천천히 원뿔을 그리며 돌았다 — 세차운동.
아무도 보지 못하는 어둠 속에서.

자기장이 있었다.
지구는 자전축에서 11.5° 기울어진 쌍극자 자기장으로 자신을 감싸고 있었다.
B ∝ 1/r³. 가까이에서만 강하고, 멀어지면 급격히 약해지는 방어막.

태양풍이 있었다.
태양에서 뿜어져 나오는 하전 입자들이 P_sw ∝ 1/r²로 공간을 채우며 밀려왔다.
자기장과 태양풍이 만나는 곳 — 7.6 R_E 지점에
보이지 않는 경계선, 마그네토포즈가 서 있었다.

그러나 —
형태가 없었다. 그림자가 없었다.
낮도 밤도 없었다. 온도도 없었다.
**빛이 없었기 때문이다.**

---

## 빛이 있으라

> 빛이 있으라 하시니 빛이 있었고
>
> And God said, "Let there be light," and there was light.
>
> — Genesis 1:3

2026-02-25, v1.3.0.

태양의 질량(1.0 M☉)으로부터 광도가 유도되었다.

```
L = M^α = 1.0^4.0 = 1.0 L☉
```

빛은 역제곱 법칙을 따라 공간으로 퍼져나갔다.

```
F(r) = L / (4πr²)
```

그 순간, 어둠 속의 질량들에 형태가 생겼다.

| 행성 | 거리 (AU) | 조도 (W/m²) | 평형 온도 (K) |
|------|----------|------------|-------------|
| 수성 | 0.387 | 9,083 | 437 |
| 금성 | 0.723 | 2,601 | 229 |
| 지구 | 1.000 | 1,361 | 254 |
| 화성 | 1.524 | 586 | 210 |
| 목성 | 5.203 | 50 | 103 |
| 토성 | 9.555 | 15 | 81 |
| 천왕성 | 19.218 | 3.7 | 58 |
| 해왕성 | 30.110 | 1.5 | 47 |

수성은 9,083 W/m²로 타올랐다.
해왕성은 1.5 W/m²의 희미한 빛을 받았다.
그 사이, 지구는 1,361 W/m²의 빛 아래에서 254 K의 온도를 갖게 되었다.

---

## 빛이 있은 후

빛이 있기 전에도 우주는 존재했다.
질량이 공간을 휘고, 중력이 궤도를 만들고,
자기장이 방패를 세우고, 태양풍이 밀려왔다.

그러나 빛이 없으면 — 형태가 없다.
형태가 없으면 — 그림자가 없다.
그림자가 없으면 — 경계가 없다.
경계가 없으면 — 존재의 의미가 없다.

빛이 켜진 순간,
수성은 타올랐고, 해왕성은 희미하게 빛났고,
지구는 254 K의 온도 위에서
자기장 방어막 안쪽에서, 태양풍을 걸러내며,
빛을 받기 시작했다.

이것이 환경의 시작이다.
이것이 존재의 시작이다.

## 엔지니어링 의미

"빛이 있으라"는 사건이 아니라 **출력의 복원**이다.

태양은 이미 주계열 항성으로 존재했지만,
v1.2.0까지 이 엔진에는 광도(L) 출력이 없었다.
중력만 있고 빛이 없는 별 — 논리적으로 불완전한 상태.

v1.3.0에서 `solar_luminosity.py`가 추가되어
질량 → 광도 → 조도 → 온도 → 복사압의 유도 체인이 완성되었다.

```
v1.2.0:  태양 = { mass: 1.0, gravity: ON, luminosity: MISSING }
v1.3.0:  태양 = { mass: 1.0, gravity: ON, luminosity: M^4.0 = 1.0 L☉ }
```

빠져 있던 출력 레이어가 채워졌다.
중력은 구조를 만들고, 빛은 환경을 만든다.
