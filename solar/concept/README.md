# concept/ — 개념 레이어 인덱스

**역할**: 우주필드 → 태양 → 태양계 → 달 → 지구 → **지구환경(1~7일)** 순서로 레이어를 **폴더로** 볼 수 있게 함.  
실제 코드는 `solar/` 루트의 `core/`, `data/`, `em/`, `atmosphere/`, `surface/`, `cognitive/`에 있고,  
**여기(concept/)에는 각 레이어별 README만** 있으며, 다음이 포함되어 있음:

- **실제 파일 구성** — 해당 레이어를 구현하는 파일 경로와 역할 표
- **엔지니어링 관점** — 환경설정 변수, 입출력, 의존 관계
- **성경 창조 의미** — 해당 “날”에 대한 짧은 의미 부여
