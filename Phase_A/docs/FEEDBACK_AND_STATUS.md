## 2. 지금 “이상한” 점 — 불일치 정리

**문서와 실제가 어긋난 부분**

- **STATUS_CHECK.md / MATH_VERIFICATION_CHECKLIST.md**  
  - "코드 통합: 미시작", "구현 직전" 으로 되어 있음.
- **실제**  
  - 이미 **코드 통합은 되어 있음**: PotentialFieldEngine에 `rotational_func` 추가, CookiieBrain에서 `enable_phase_a`로 연동, `phase_a_integration_test.py` 존재.

**순서 불일치**

- 권장 순서: **부호·직교성 수치 검증** → 그 다음 구현.
- 실제 진행: **구현을 먼저** 했고, **수치 검증은 아직 안 함.**

**정리**:  
설계·구현은 들어가 있지만, **“구현 직전”** 이라는 문서 표현과 **“검증 미실행”** 상태가 맞지 않아서 이상하게 느껴지는 상태다.
