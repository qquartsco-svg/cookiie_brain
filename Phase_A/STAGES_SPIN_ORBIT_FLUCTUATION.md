# 단계 정리: 자전 → 공전 → 요동 (개념만)

**원칙**: 감정 없음. 개념·층위·순서만 기록.

---

## 결론

**자전 → 공전 → 요동** 은 "물리적 발전 단계"가 아니다.  
**서로 다른 층위**다.

- 자전·공전: **구조를 복잡하게 만드는 것** (고전 확장)
- 요동: **구조의 안정성을 깨는 것** (결정론 위에 확률 층)

→ **고전 구조 확장 → 그 위에 확률적 층 추가** 에 가깝다.

---

## 1. 자전

| 항목 | 내용 |
|------|------|
| 정의 | 한 중심을 기준으로 한 내부 회전 |
| 수식 | 회전항 ωJx (또는 ωJ(x−x_pole)) |
| 성질 | 닫힌 궤도 가능, 완전 결정론적 |
| 층위 | **고전적 위상 구조 생성** |

---

## 2. 공전

| 항목 | 내용 |
|------|------|
| 정의 | 두 개 이상의 질량 중심 상호작용, 중심 이동 |
| 성질 | 중력장(필드) 자체가 위치에 따라 변함, 여전히 결정론적 |
| 층위 | **다체 상호작용 구조** |

---

## 3. 요동

| 항목 | 내용 |
|------|------|
| 정의 | 확률성, 최소점 고정 불가, 진공에서도 흔들림 |
| 기반 | 불확정성 원리 |
| 층위 | **결정론을 깨는 층** (다른 축) |

---

## 더 정확한 순서

| 단계 | 내용 |
|------|------|
| 1 | 정적 포텐셜 (현재 상태) |
| 2 | 회전항 추가 (자전) |
| 3 | 다중 중심 상호작용 (공전) |
| 4 | 확률 요동 추가 |

**요동은 항상 마지막에 얹는 것이 안정적.**

---

## 왜 요동이 마지막인가

- 회전 없이 요동만: 그냥 랜덤 워크.
- 공전 없이 요동만: 복잡한 확률 궤도 해석 어려움.
- **구조가 먼저 있어야 요동의 의미가 생긴다.**

## 다음 설계 핵심 (미결정)

**자전을 어떤 단위로 둘지:**

| 옵션 | 설명 |
|------|------|
| **단일 중심 회전** | 하나의 전역 중심(예: 원점) 기준 ωJ(x−x_pole). |
| **우물 중심 주변 국소 회전** | 우물 바닥(또는 각 우물 중심)을 기준으로 한 국소 회전. 우물마다 폴이 다를 수 있음. |

→ 이 선택이 다음 설계 핵심.
