# 인지 태양계 — Cognitive Solar System

## 한 문장 요약

> 실제 태양계의 N-body 역학(공전, 자전, 세차운동)을 수치적으로 구현하고,
> 각 천체를 인지 기능에 매핑하여 "기억이라는 거대질량을 중심으로 하는 인지 조화 시스템"을 만든다.

## 1. 핵심 개념: 보이지 않는 기어

태양계의 행성들은 물리적으로 떨어져 있다.
하지만 **중력 필드장**이라는 보이지 않는 연결로 서로 맞물려 돌아간다.

```
수성 ←─── 중력장 ───→ 금성
 ↕                       ↕
태양 ←── 중력장 ──→ 지구 ←── 중력장 ──→ 달
 ↕                       ↕
화성 ←─── 중력장 ───→ 목성
```

각 행성은:
- **자전** (자기 축으로 회전)
- **공전** (태양 주위 궤도)
- **세차운동** (자전축이 기울어진 채로 팽이처럼 흔들림)

이 세 가지 회전이 중력 필드장으로 연결되어 **기어처럼 맞물린다**.
한 행성의 움직임이 다른 행성에 섭동(perturbation)을 주고,
그 섭동이 다시 되돌아온다.

**이것이 엣지 AI 시스템이다.**

각 노드(행성)는 자체적으로 처리(자전)하면서,
필드장(중력)으로 연결되어 전체 시스템이 조화를 이룬다.
중앙 서버(태양)가 있지만, 각 노드가 자율적으로 동작한다.

### 2.3 핵심 수식

**N-body 중력:**
```
F_i = Σ_{j≠i} G·m_i·m_j·(r_j - r_i) / |r_j - r_i|³
```

**세차운동 (지구 자전축):**
```
지구 자전축: ŝ (기울기 23.4°)
태양/달이 적도 부풀음에 가하는 토크:
  τ = (3GM/r³) · C₂ · (ŝ × r̂)(ŝ · r̂)
  C₂ = (C - A)/C ≈ 0.00327  (지구 편평도)

세차 주기: ~25,772년
장동 주기: ~18.6년 (달 궤도 교점 주기)
```

**자전:**
```
θ_spin(t) = ω_spin · t
ω_spin = 2π / T_rotation
```

## 4. 인지 매핑: 각 행성 = 인지 기능

### 4.1 태양계 → 인지 시스템 대응표

|------|-----------|------|-------------|

| **수성** | 즉각 반사 | 태양에 가장 가까움, 빠른 공전 | 0.24년 주기 |
| **금성** | 감정/정서 | 역자전(다른 모든 것과 반대), 두꺼운 대기 | 역방향 자전 |
| **지구** | 현재 의식 | 생명, 물(우물/바다), 자전축 세차 | 유일한 액체 물 |

| **목성** | 보호 기제 | 소행성 흡수, 거대 질량으로 보호 | 소행성 포획 |
| **토성** | 구조화/범주화 | 고리(경계), 정돈된 구조 | 고리 시스템 |

### 4.2 기어 맞물림 = 인지 상호작용

```
태양(장기기억) ←─ 강한 중력 ─→ 수성(즉각 반사)
  기억에 가장 가까운 반응. 빠르게 순환.

지구(의식) ←─ 조석 ─→ 달(무의식)
  의식과 무의식이 서로 끌어당기며 리듬을 만든다.
  세차운동 = 의식의 장기적 방향 변화 (25,772년 주기)

목성(보호) ←─ 섭동 ─→ 소행성대
  외부 충격(trauma)을 흡수하여 내부 궤도를 보호.

금성(감정) ←─ 공명 ─→ 지구(의식)
  감정과 의식의 공명. 금성의 역자전 = 감정은 이성과 반대로 움직인다.
```

### 4.3 세차운동의 인지적 의미

```
지구 자전축: 23.4° 기울어짐
  → 계절이 생김 (기울기가 없으면 계절 없음)
  → 인지: "관점의 기울기"가 있어야 다양한 경험(계절)이 가능

세차운동: 자전축이 25,772년 주기로 원을 그림
  → 북극성이 바뀜
  → 인지: "기준점(북극성)"이 서서히 이동
  → 장기적으로 가치관, 세계관이 천천히 변화

장동: 세차 위에 18.6년 주기 진동
  → 달 궤도면의 요동
  → 인지: 무의식(달)의 리듬이 의식(지구)의 방향을 미세하게 흔듬
```

### 6.1 레이어 확장

```
  L0 State:    (x₁,v₁, x₂,v₂, ..., xₙ,vₙ, ŝ₁,...,ŝₙ) — N개 천체, 3D
  L1 Dynamics: N-body symplectic 적분기
  L2 Field:    N-body 중력 + J2 편평도 + 조석
  L2.5 Spin:   세차/장동 토크 → 스핀 벡터 적분
```

### 6.2 CelestialBody 클래스 (제안)

```python
@dataclass
class CelestialBody:
    name: str
    mass: float
    position: np.ndarray     # 3D
    velocity: np.ndarray     # 3D
    spin_axis: np.ndarray    # 3D 단위벡터 (자전축 방향)
    spin_rate: float         # 자전 각속도
    obliquity: float         # 자전축 기울기 (rad)
    J2: float = 0.0          # 편평도 (세차운동용)
    radius: float = 0.0      # 물리 반지름 (J2 토크 계산용)

    # 인지 매핑
    cognitive_role: str = ""  # "memory_core", "consciousness", etc.
    well_amplitude: float = 0.0  # 이 천체가 만드는 우물 깊이
```

### 6.3 SolarSystemEngine 클래스 (제안)

```python
class SolarSystemEngine:
    """N-body 태양계 + 세차운동 + 인지 매핑"""

    def __init__(self, bodies: List[CelestialBody]):
        self.bodies = bodies
        self.n = len(bodies)
        self.masses = np.array([b.mass for b in bodies])

    def update(self, dt: float):
        # 1. N-body 중력 계산 (모든 쌍)
        forces = self._compute_gravity()

        # 2. 위치/속도 적분 (symplectic leapfrog)
        self._leapfrog_step(forces, dt)

        # 3. 세차운동 토크 계산 + 스핀 벡터 적분
        self._precess_spins(dt)

        # 4. 인지 상태 갱신 (우물 위치 = 행성 위치)
        self._update_cognitive_state()
```

## 7. 엣지 AI 시스템으로서의 의미

### 7.1 왜 "엣지 AI"인가

```
중앙 집중형 AI:
  하나의 거대 모델이 모든 것을 처리
  → 태양만 있는 시스템 (행성 없음)

엣지 AI:
  각 노드가 자체적으로 처리 + 서로 통신
  → 태양계: 각 행성이 자전(자체 처리)하면서
    중력장(통신)으로 연결되어 전체가 조화

핵심 차이:
  - 각 노드(행성)는 자율적 (자전)
  - 통신은 필드장 (중력) — 직접 연결이 아닌 공간을 통한 간접 상호작용
  - 거리에 따라 영향력 감소 (1/r²)
  - 하지만 완전히 끊어지지 않음 (중력은 무한히 도달)
```

### 7.2 기어 맞물림 메커니즘

```
물리적 기어: 접촉해야 맞물림
중력 기어:   떨어져 있어도 맞물림

목성 ←──── 5.2 AU ────→ 태양
  물리적 접촉: 없음
  중력 접촉: 있음 (1/r²로 감소하지만 0이 아님)
  효과: 태양이 미세하게 흔들림 (바리센터 이동)

이것이 "보이지 않는 기어"다.
각 행성의 자전(처리)이 공전(궤도)을 통해
다른 행성의 궤도에 섭동을 주고,
그 섭동이 다시 자신에게 돌아온다.

인지 시스템에서:
  장기기억(태양)의 변화 → 전체 인지 궤도가 미세하게 조정
  무의식(달)의 리듬 → 의식(지구)의 조석 → 감정(금성)의 공명
  외부 충격 → 보호 기제(목성)가 흡수 → 내부 궤도 안정 유지
```

## 9. 핵심 질문과 답

**Q: 세 포인트만으로 안 되면?**
A: N-body로 확장하면 됩니다. 8개 행성 + 달 = 10체 문제.
   중력 계산은 O(N²)이지만, N=10이면 100번의 쌍 계산. 가볍습니다.

**Q: 세차운동까지 구현 가능한가?**
A: 네. 토크 수식이 알려져 있고, 스핀 벡터 적분만 추가하면 됩니다.
   3D 확장이 필요하지만, 현재 2D 코드의 자연스러운 확장입니다.

**Q: 실제 데이터로 검증할 수 있는가?**
A: 네. NASA JPL Horizons에서 정밀 궤도 데이터를 가져올 수 있고,
   이미 실제 비율로 20/20 PASS를 확인했습니다.

**Q: 이게 왜 "인지 시스템"인가?**
A: 태양계와 인지 시스템이 **같은 동역학 구조**를 공유하기 때문입니다.
   N-body 중력 = 기억/인지 요소 간 상호작용
   공전 = 인지 상태 순환
   세차 = 장기적 관점 변화
   수학이 같으면, 한쪽에서 증명된 것이 다른 쪽에서도 성립합니다.
