## 3. 구조 요약 (레이어·의존 방향)

```
data/ → core/ ← em/
              ← atmosphere/  ← biosphere/
              ← surface/        │
                                ├ 읽기: F, T, P, CO2, H2O, O2, water_phase, land_fraction
                                └ 쓰기: delta_CO2, delta_O2, transpiration, latent_heat, delta_albedo_land
```

- **surface/**: 땅·바다 분리, 유효 알베도 (Phase 7A)
- **biosphere/**: 풀·씨·열매·나무 (Phase 7B). atmosphere·surface와만 연결. core/ 건드리지 않음.
- **brain_core_bridge**: `get_solar_environment_extension(..., surface_schema, biospheres)` 호출 시, Earth에 대해 biosphere.step() → feedback → composition/albedo/잠열 반영.
