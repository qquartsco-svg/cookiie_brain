## 6. Phase 7b: 토양 형성 ODE (셋째날 — 선구 생물)

**세션**: 2026-02-26 오후

### 6.1 핵심 설계 철학 (세차운동과 동일)

세차운동 방식:
- 입력: G, M, r (물리 상수·관측값)
- 결과: 25,000년 주기가 자연스럽게 나온다

토양 형성 동일 방식:
- 입력: R, W, ETA, λ (물리 법칙·관측값)
- 결과: 2739년 후 원시토양 완성이 자연스럽게 나온다

### 6.2 ODE 시스템 (3변수)

```
dP/dt = R·P·(1 - P/K(S))·fT·fW - M·P      [로지스틱 성장]
dM/dt = W_w·P·fT·fW                          [풍화: pioneer → 광물]
dS/dt = ETA·M·P + W_mh·M_min - λ(T)·S       [humus 축적, Q10 분해]

양성 피드백: K(S) = K0 + K_soil·S
Q10 분해:   λ(T) = λ_base × 2^((T-283)/10)
```

## 7. Phase 7c: 식물 생애주기 Phase Gate ODE (v1.8.0)

**세션**: 2026-02-26 오후 (session 3)

### 7.1 배경 — 기존 코드의 문제점

| 문제 | 내용 |
|------|------|
| dt 이중 적용 | `to_seed = K×B×dt` 후 `B += to_seed×dt` → dt² 오류 |
| allocation 미정규화 | a_leaf+a_root+a_wood+a_seed ≠ 1 → 탄소 회계 불일치 |
| 생애주기 단계 없음 | B_leaf/root/wood/seed 하나의 pool, 순서(씨→싹→줄기→나무→열매) 없음 |

### 7.2 핵심 수정

**dt 수정**: 모든 변화량을 율(d/dt) [단위/yr]로 정의 후 dt 한 번만 적용
```
수정 전: to_seed = K×B×dt  →  B += to_seed×dt     (dt²)
수정 후: d_rate  = K×B      →  B += d_rate×dt      (dt¹)
```

**softmax allocation**: logits(O₂, phase) → softmax → sum(alloc)=1 보장
```python
logits = {leaf: base+Δ(O₂), root: base, stem: base×(1-O₂), wood: base+Δ(O₂), fruit: base+Δ(O₂)}
alloc  = softmax(logits)   # 항상 합=1
```

**Phase Gate ODE**: sigmoid Michaelis-Menten 게이트 (if-else 없음, 연속 미분)
```
B_seed   → K_germ × g_soil(S) × g_T × f_W             → B_sprout  [발아]
B_sprout → K_spr × S(B_sp, B_sp_half)                  → B_stem    [유묘→줄기]
B_stem   → K_stw × S(B_st, B_st_half)                  → B_wood    [목질화]
B_wood   → K_fruit × S(B_wd, B_wd_th) × f_O2           → B_fruit   [결실]
B_fruit  → K_fts                                        → B_seed ↺  [순환]
```
