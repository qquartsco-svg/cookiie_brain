## 6. Phase 7b: 토양 형성 ODE (셋째날 — 선구 생물)

**세션**: 2026-02-26 오후

### 6.1 핵심 설계 철학 (세차운동과 동일)

세차운동 방식:
- 입력: G, M, r (물리 상수·관측값)
- 결과: 25,000년 주기가 자연스럽게 나온다

토양 형성 동일 방식:
- 입력: R, W, ETA, λ (물리 법칙·관측값)
- 결과: 2739년 후 원시토양 완성이 자연스럽게 나온다

### 6.2 ODE 시스템 (3변수)

```
dP/dt = R·P·(1 - P/K(S))·fT·fW - M·P      [로지스틱 성장]
dM/dt = W_w·P·fT·fW                          [풍화: pioneer → 광물]
dS/dt = ETA·M·P + W_mh·M_min - λ(T)·S       [humus 축적, Q10 분해]

양성 피드백: K(S) = K0 + K_soil·S
Q10 분해:   λ(T) = λ_base × 2^((T-283)/10)
```
