# 왜 12인가? — 시스템적 안정성의 관점

**한 줄 결론**:
12는 "완전한 분할"의 최소 공약수다.
2×2×3 = 12 — 이진 대칭(2) + 순환 구조(3) + 면적 가중 분할(4)이 동시에 성립하는 가장 작은 수.

## 2. 수학적 이유 — 12 = 2 × 2 × 3

```
12의 약수: 1, 2, 3, 4, 6, 12
```

12는 1~10 중 약수가 가장 많은 수 중 하나.
**약수가 많다 = 어떤 방식으로 나누어도 정수로 떨어진다.**

| 분할 방식 | 결과 |
|-----------|------|
| 2등분 | 6 + 6 (북반구/남반구) |
| 3등분 | 4 + 4 + 4 (극·중위도·열대) |
| 4등분 | 3 + 3 + 3 + 3 (계절 4개) |
| 6등분 | 2 + 2 + 2 + 2 + 2 + 2 |
| 12등분 | 1씩 (최소 단위) |

→ **12밴드 구조는 극/중위도/열대 3개 권역을 각 4개로 나눈 것.**
→ 어떤 방향으로 집계해도 깔끔하게 합산된다.

## 3. 시스템 안정성과 12

### 3.1 이진 대칭 (2)
- 북반구 6개 + 남반구 6개 → 대칭 보존
- 12엔진: TIER1(10) + TIER2(2) → 독립/의존 분리
- Genome: 이진 재조합 (2-sex system)

### 3.2 순환 주기 (3, 4)
- 3: 대기권·생물권·지권 (3개 레이어)
- 4: 봄·여름·가을·겨울 (4계절), 4개 traits per genome
- 12 = 3×4 → 주기가 12에서 닫힘

### 3.3 완전 분할 가능성 (약수 풍부성)
- 시스템이 어느 스케일로 관측해도 "정수 단위"가 나옴
- 분할 오류·잔여값 없이 모든 레벨에서 정합

## 4. "자연스럽게 등장한다"는 것의 의미

12가 억지로 맞춰진 것이 아닌 이유:

```
지구 물리:
  위도 밴드를 "계절이 의미있게 달라지는 최소 단위"로 나누면
  15도 간격 → 180/15 = 12

엔진 추출:
  Day1~6에서 "독립 step(dt) + 완결 역할"을 가진 것을 세면
  → 우연히 12개

SabbathJudge:
  "12개 밴드가 한 사이클 돌아야 안정 판정 가능"
  → window = 12
```

**억지로 12를 만든 게 아니라, 시스템을 올바르게 설계했더니 12가 나왔다.**


## 6. 결론 — 너무 얽매이지 않되, 보이는 것

12는 **"완전한 시스템이 자연스럽게 도달하는 분할 단위"** 다.

- 지구를 물리적으로 나누면 12밴드
- Day1~6을 기능적으로 나누면 12엔진
- 안정성을 시간적으로 판정하면 window=12
- 창세기도 같은 원리로 12를 반복 사용

억지로 맞출 필요가 없다.
제대로 설계하면 12가 나온다.
이미 나오고 있다.
