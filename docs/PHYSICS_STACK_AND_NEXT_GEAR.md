### Phase 6b: 수순환 + 잠열 ✅ (v1.5.0)

| 물리 | 수식/역할 |
|------|-----------|
| Clausius-Clapeyron | e_sat(T), q_sat(T,P) |
| 증발율 | E = C_E·ρ·U·(q_sat - q) |

**의미**: 바다 ↔ 대기 질량·에너지 교환 루프, **거시적 호흡** 가동.

## 2. 동역학 시스템 관점 — 현재 구조의 성격

### 수학적 형태

```
C · dT/dt = F(t) - σT⁴(1 - ε/2) - Q_latent
```

- **외부 강제력(forced)** + **1차 완화계(thermal relaxation)**
- 단일 안정 평형점
- 카오스 없음, 창발 없음

### 결론

| 속성 | 상태 |
|------|------|
| 물리적 일관성 | ✅ |
| 수치 안정 | ✅ |
| 보존 깨지지 않음 | ✅ |
| 창발 | ❌ |
| 카오스 | ❌ |
| 자가조직화 | ❌ |

→ **"정지한 지구"** — 환경 완충 시스템. 아직 "자가 진화 단계" 아님.

### 창발이 생기려면

- **(A) 비선형성** — 이미 있음 (T⁴)
- **(B) 피드백 루프** — 지금은 약함
- **(C) 지연/메모리** — 열용량으로 약간 존재
- **(D) 2개 이상 상태변수 상호작용** — 지금은 거의 T 1변수

**필요한 구조 예시**:
- T↑ → H₂O↑ → τ↑ → ε↑ → T↑ (양의 피드백)
- T↑ → 증발↑ → 구름↑ → 알베도↑ → F↓ → T↓ (음의 피드백)
- 이 두 개가 같이 들어가면: 다중 평형, Hopf bifurcation, limit cycle 가능.

## 3. 다음 물리 기어 후보 — "기어 맞춰보기" 우선순위

목적: **상태공간이 어떻게 변하는지 관찰**.

### 환경 안정성 기준 (장기 관점)

| 순위 | 기어 | 물리 법칙 | 의미 | 영향 |
|------|------|-----------|------|------|
| 1 | **대기 탈출** | v_thermal = √(2kT/m), λ = GMm/(kTR) | λ < ~10이면 탈출 심각 | ★★★★ |
| 2 | **수증기 피드백 강화** | τ(H₂O)를 T에 종속 | 이미 6b에 토대 있음 | ★★★ |
| 3 | **지각/맨틀 열유속** | F_geo = Q_int/(4πR²) | Outgassing, CO₂ 공급 | ★★★ |
| 4 | **Outgassing** | 맨틀 → CO₂, H₂O | 조성 변화 | ★★★ |
| 5 | **ice-albedo** | 알베도 A(T) | 양의 피드백 | ★★★ |
| 6 | **광화학** | UV → O₃ | O₃/UV 차폐 | ★★ |

### 창발/피드백 루프 기준 (시스템 살리기)

다음 중 **하나만** 넣어도 시스템이 "살아남":

1. **H₂O를 T에 종속** (Clausius-Clapeyron) — 6b에 이미 부분 구현
2. **알베도를 T에 종속** (ice-albedo feedback)
3. **CO₂를 outgassing vs weathering로 동역학화**
4. **태양광도를 시간 함수** F(t)
5. **궤도 이심률 진동** (Milanković)

### Phase 7 후보 (용자 제안)

- **원시 대사 (Proto-Metabolism)**: 수순환 + 대기순환 → 와도 고착 → "호흡하는 세포"
- **Outgassing**: 지각 활동 → CO₂ 방출 → 내부 추동력
- **Hadley Cell**: 0D → 위도별 확장 → 전역 순환

## 4. 큰 그림 — 환경 설정 vs 지속가능성

| 구분 | 상태 |
|------|------|
| 환경 설정 | **~70%** 완성 |
| 지속 가능성 (대기 탈출 등) | 미완 |
| 비선형 피드백 | 미완 |
| 창발/자가조직화 | 미완 |

**비유**: 크랭크축은 돌아가고, 피스톤 하나는 있다. 아직 밸브가 없다.

## 5. 다음 단계 — 어디부터 파고들까

**목표**: 물리 법칙을 대입해 나가면서, 어떤 상황·코드 상태가 펼쳐지는지 보기.

### 추천 순서 (실험 용이성 기준)

1. **수증기 피드백 강화** — 6b 확장, τ(H₂O)를 T·q에 강하게 연결
2. **ice-albedo feedback** — A(T) 도입, 양의 피드백 테스트
3. **대기 탈출** — λ = GMm/(kTR) 판정, 장기 안정성
4. **Outgassing** — 단순 F_geo + composition 갱신 (지권 미구축이라 파라메트릭)
5. **Hadley 단순화** — 0D 유지하면서 "유효 바람계수" 형태로 근사
