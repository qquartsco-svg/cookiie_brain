# 자가순환 구조의 물리적 조건

**목적**: 물리 법칙을 공간에 하나씩 대입했을 때,  
**외부 강제력에 끌려다니는 물질**이 아니라  
**스스로 순환·유지되는 구조**가 나타나는 조건을 정리한다.

---

## 1. 현재: 수동 시스템

구현된 것들은 모두 **외부 에너지 입력에 대한 반응**이다.

| 물리 | 역할 |
|------|------|
| 중력 | 위치 변화 |
| 복사 | 가열 |
| 온실 | 재분배 |
| 자기장 | 차폐 |
| 열용량 | 지연 |

수학적 형태:
```
ẋ = f(x) + F_external
```
→ **외력에 반응하는 완화계**. 아직 "자가순환" 아님.

---

## 2. 자가순환 구조의 최소 4조건

### (A) 에너지 구배 ✅
- 태양 복사, 화산 열, 전기화학 구배  
- **이미 있음.**

### (B) 비평형 상태 ⚠️
- 평형 도달 시 운동은 멈춤.
- 자가순환은 반드시: **에너지 유입 ≠ 에너지 소산**
- 현재: 거의 복사 평형 수렴 → 비평형이 약함.

### (C) 비선형 반응 + 폐루프 ❌
- 화학 루프: A → B → C → A  
- 피드백 루프: T↑ → 증발↑ → 구름↑ → 알베도↑ → T↓  
- **현재 코드: 이런 폐루프 없음.**

### (D) 경계 조건 ❌
- 세포막, 소용돌이 경계, 열 대류 셀 경계, 화학 농도 경계  
- 경계 없으면 에너지 확산으로 구조 유지 불가.  
- **현재 코드: 경계 형성 메커니즘 없음.**

---

## 3. "스스로 움직이는 객체"의 열역학적 정의

중력·전자기력에 끌려가는 게 아니라,  
**내부 에너지 흐름**으로 상태가 유지·갱신되는 구조.

열역학:
```
dS/dt_internal < 0    (국소 엔트로피 감소)
dS/dt_total    > 0    (전체는 증가)
```
→ 생명의 최소 조건.

---

## 4. 지구 환경에서 가능한 출발점

| 출발점 | 특성 |
|--------|------|
| **해저 열수구** | 화학 구배, pH 구배, 전기화학 구배, 미네랄 표면 촉매 |
| **대기-해양 경계** | UV 분해, 번개, 습윤-건조 사이클 |
| **대류 셀** | Rayleigh-Bénard, 자가 유지 소용돌이 |

---

## 5. 코드 상태공간에서 빠진 것

**있는 것**  
- 열 ODE, 복사, 압력, 자기장  

**없는 것**  
- ❌ 화학 농도 상태변수  
- ❌ 반응 속도식  
- ❌ 확산 방정식  
- ❌ 경계 형성 메커니즘  

→ **아직 "물질 동역학"이 없다.**

---

## 6. 자가순환의 최소 수학 구조

최소 형태:
```
Ȧ = f(A, B)
Ḃ = g(A, B)
```

요구사항:
- 비선형
- 폐루프
- 외부 에너지 입력

예:
- Brusselator  
- Lotka-Volterra  
- Belousov–Zhabotinsky  

---

## 7. 결론

| 구분 | 상태 |
|------|------|
| **환경 설정** | ✅ 완료 (바다, 대기, 태양, 자기장) |
| **물질 반응 동역학** | ❌ 미구축 |

- 단순 역학 + 복사 + 온실만으로는 **자가순환 객체는 나오지 않는다**.
- **화학 반응 + 구배 + 확산 + 경계 형성**이 들어가야 나온다.
- 그 시점부터 상태공간이 본격적으로 복잡해진다.

### 현실 점검: ODE만으로는 부족

반응 ODE만 넣으면 **불충분**:
- 경계 없으면 농도는 확산으로 평탄화 → 자가순환 유지 불가
- **반응-확산 PDE** 구조 필요 (공간 격자 + ∇²c 항)
- 화학 창발은 확률적, 초기 조건 민감, 공간 불균일성 필수

- 기어 분리 규칙을 유지하면서 화학·확산·경계를 어디에 넣을지  

이를 개념적으로 정리해야 한다.
