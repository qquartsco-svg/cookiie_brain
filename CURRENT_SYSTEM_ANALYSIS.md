# 현재 시스템 분석: 왜 진동이 없는가?

## 🔍 현재 시스템 구조

### 이미 구현된 것

**PotentialFieldEngine의 업데이트**:
```python
# 속도 업데이트
v_new = v + self.dt * a  # a = g = -∇E(x)

# 위치 업데이트
x_new = x + self.dt * v_new
```

**상태 벡터 구조**:
```python
state_vector = [x1, x2, ..., xN, v1, v2, ..., vN]
```

**에너지 계산**:
```python
energy = kinetic_energy + potential_energy
       = (1/2) * v^2 + V(x)
```

## ⚠️ 왜 진동이 발생하지 않는가?

### 문제점: 완전 감쇠 (Overdamped)

**현재 업데이트 수식**:
```
v_{t+1} = v_t + dt * (-∇E(x_t))
x_{t+1} = x_t + dt * v_{t+1}
```

**분석**:
- 이건 **2차 동역학**이지만
- **감쇠 항이 없음** (γ = 0)
- 하지만 **에너지 주입도 없음**
- 결과: **완전 감쇠** (에너지가 계속 소모됨)

**물리적 의미**:
- 마찰이 없는 이상적인 시스템
- 하지만 초기 에너지만 있고 추가 에너지 없음
- → 최소점으로 수렴하면서 운동 에너지 소모
- → 결국 정지

---

## 🎯 진동이 발생하려면

### 필요한 조건

#### ① 감쇠 조정 (Damping Control)

**현재**: 감쇠 없음 (γ = 0, 암묵적)

**필요**: 감쇠 항 추가
```
v_{t+1} = v_t + dt * (-∇E(x_t) - γ * v_t)
```

**효과**:
- γ > 0: 감쇠 (진동 감소)
- γ < 0: 음의 감쇠 (진동 증가, 불안정)
- γ = 0: 현재 상태 (에너지 소모만)

#### ② 에너지 주입 (Energy Injection)

**방법 1**: 주기적 외부 자극
```
v_{t+1} = v_t + dt * (-∇E(x_t) + F_ext(t))
```

**방법 2**: 내부 발진기 (Oscillator)
```
v_{t+1} = v_t + dt * (-∇E(x_t) + F_osc(x, v))
```

#### ③ 비선형 진동 구조

**방법**: Limit Cycle 생성
- 최소점 대신 궤도가 안정 구조
- Hopf bifurcation

## 📊 현재 vs 목표

### 현재 시스템

**특징**:
- ✅ 2차 동역학 구조 (위치 + 속도)
- ✅ 관성 개념 포함
- ❌ 완전 감쇠 (진동 없음)
- ❌ 에너지 주입 없음
- ❌ 위상 구조 없음

**결과**: 최소점으로 수렴 후 정지

### 목표 시스템 (자전)

**필요한 것**:
- ✅ 2차 동역학 구조 (이미 있음)
- ⚠️ 감쇠 조정 또는 에너지 주입
- ⚠️ 위상 구조 추가
- ⚠️ 주기성 생성

**결과**: 최소점 주변에서 진동 (자전)

---

## 🔧 구현 방향

### 옵션 1: 감쇠 조정

**간단한 방법**:
- 감쇠 항 추가
- 약한 감쇠로 설정
- → 미세 진동 유지

### 옵션 2: 에너지 주입

**복잡한 방법**:
- 외부 자극 또는 내부 발진기
- 에너지 주입 + 소산 균형
- → 지속적 진동

### 옵션 3: 비선형 진동

**가장 복잡한 방법**:
- Limit Cycle 구조
- Hopf bifurcation
- → 복잡한 리듬
