# CookiieBrain — 레이어 흐름

---

## 레이어 흐름 (번호순)

```
┌─────────────────────────────────────────────────────┐
│  Layer 0 — 서사                                      │
│  L0_solar/                                           │
│  천지창조 day1~7 → 에덴·OS·언더월드 → 노아 대홍수    │
└──────────────────────┬──────────────────────────────┘
                       │  cookiie_brain_engine.py (통합·연결)
                       ↓
┌─────────────────────────────────────────────────────┐
│  Layer 1~2 — 동역학                                  │
│  L1_dynamics/                                        │
│  Phase_A : 자전·조석·코리올리 (필드 회전)             │
│  Phase_B : 우물→가우시안 (에너지 지형)               │
│  Phase_C : 요동·FDT (확률적 진동)                    │
└──────────────────────┬──────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  Layer 3 — 기억                                      │
│  L3_memory/                                          │
│  우물 생성·강화·감쇠·소멸 (MemoryStore)               │
│  탐색 / 정착 / 리콜 (EnergyBudgeter)                 │
└──────────────────────┬──────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  Layer 4 — 분석                                      │
│  L4_analysis/                                        │
│  Layer_1~6: 궤적·물리량 측정. 시스템 변경 안 함.      │
└─────────────────────────────────────────────────────┘
```

---

## 각 레이어가 하는 일

| # | 이름 | 폴더 | 하는 일 |
|---|------|------|---------|
| **0** | 서사 | `L0_solar/` | 서사 실행. 천지창조·에덴·언더월드·노아가 여기서 돌아감. |
| **—** | 통합 | `cookiie_brain_engine.py` | 레이어 0~4 전부 연결하는 오케스트레이터. |
| **1** | 필드 회전 | `L1_dynamics/Phase_A/` | 자전·조석·코리올리 힘. 회전하는 장(field). |
| **2** | 에너지 지형 | `L1_dynamics/Phase_B/` | 우물→가우시안. 입자가 굴러다닐 지형을 만듦. |
| **2** | 요동 | `L1_dynamics/Phase_C/` | 열적 요동·FDT. 확률적 진동 주입. |
| **3** | 기억 | `L3_memory/` | 우물을 동적으로 만들고·키우고·지움. |
| **4** | 분석 | `L4_analysis/` | 궤적을 측정·분석. 시스템은 건드리지 않음. |
