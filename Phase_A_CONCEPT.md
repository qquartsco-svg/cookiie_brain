## 🎯 핵심 개념

### 위상 생성이란?

**의미**: 순환 운동을 가능하게 하는 회전 성분 추가

**수학적 표현**:
```
기존: g = -∇E  (curl = 0, 순환 불가능)
목표: g = -∇E + R  (curl(R) ≠ 0, 순환 가능)
```

---

## 🔄 Rotational Component

### 물리적 의미

**회전 성분**:
- 중심점(폴) 주변을 회전하는 힘
- curl ≠ 0인 비보존력
- 순환 운동의 원천

**비유**:
- 소용돌이의 중심
- 회전하는 물의 흐름
- 자전의 원동력

---

## 📍 폴(Pole)의 역할

### 폴이란?

**정의**: 회전의 중심점

**역할**:
1. 회전의 기준점
2. Rotational field의 중심
3. 자전 축의 기준

**위치 결정**:
- 에너지 최소점 (우물 중심)
- 임의 지정
- 동적 계산

---

## 🌙 달/위성의 역할

### 물리적 배경

**지구-달 시스템**:
- 지구의 자전은 달의 조석력에 영향받음
- 달은 자전을 안정화/조절
- 자전 자체를 만드는 것은 아님

---

### 수학적 분석

#### 자전만 필요한 경우

**필요한 것**:
```
F(x) = -∇E(x) + R(x)  (curl(R) ≠ 0)
```

**결론**: 달/위성 없이도 자전 가능

---

#### 달/위성이 추가되면?

**효과**:
1. **조석력**: 자전 속도 조절
2. **안정화**: 자전 축 안정화
3. **복잡한 동역학**: 더 풍부한 패턴

**필드 정의**:
```
g(x) = -∇E(x) + R(x) + G_moon(x)
```

**결론**:
- 자전을 만드는 필수 요소 아님
- 자전을 안정화/조절하는 데 유용
