# Layer 6 — 정보 기하학 (Information Geometry)

매개변수 공간의 기하학적 구조를 Fisher 정보 계량으로 분석한다.

## 위치

```
Layer 3: 물리 공간 게이지 — Ω(x)v, B-field (공간 내 기하)
Layer 6: 정보 기하학    — g_μν, K   (매개변수 공간 기하)
```

Layer 3이 **물리 공간의 게이지 구조**라면,
Layer 6은 **매개변수 공간의 Riemannian 기하학**이다.

---

## 핵심 수식

### Fisher 정보 계량

ρ(x; λ) ∝ exp(−V(x;λ)/T) 일 때:

```
g_μν(λ) = ∫ ρ(x;λ) · ∂_μ ln ρ · ∂_ν ln ρ dx
         = (1/T²) · Cov_λ(∂_μV, ∂_νV)
```

- **양정치(positive definite)**: 분포가 축퇴하지 않으면 g > 0.
  매개변수가 분포에 영향을 주지 않는 singular case에서는 semi-definite (det(g) = 0).
- **대칭**: g_μν = g_νμ
- **통계적 거리**: ds² = g_μν dλ^μ dλ^ν = "분포 변화의 크기"

### 가우스 곡률

```
K = [det(A) − det(B)] / det(g)²     (Brioschi 공식)
```

- K > 0: 구면 유사 (매개변수 간 "수렴")
- K < 0: 쌍곡면 유사 (매개변수 간 "발산")
- K = 0: 평탄 (유클리드)

### 측지선 거리

```
L = ∫ √(g_μν dλ^μ dλ^ν)
```

매개변수 변화의 "통계적 비용".
L_Fisher ≥ L_Euclid는 **g_μν가 δ_μν보다 큰 경우**에만 성립한다.
일반적으로는 metric scale에 의존하며, 보편적 부등식이 아니다.

---

## 왜 naive Berry phase = 0인가 (1D 고전)

```
A_μ = ∂⟨x⟩/∂λ_μ = ∇_λ f   (gradient of a scalar)
F = curl(∇f) = 0             (항상, Schwarz 정리)
```

1D overdamped 고전계에서 스칼라 기대값의 Berry connection은 trivial하다.
이것은 버그가 아니라 **수학적 사실**이다:
- 양자: |ψ⟩에 복소 위상 자유도 → U(1) fiber bundle → 비자명 Berry 위상
- 고전: ρ(x;λ)는 실수 양의 확률 → gradient connection → trivial

Fisher 계량은 이와 **독립적**으로 비자명한 기하학을 제공한다.

## 양자 대응

| 양자 | 고전 (본 구현) |
|------|---------------|
| Fubini-Study 계량 | Fisher 정보 계량 |
| Berry connection A_μ | (trivial: curl = 0) |
| Berry curvature F_μν | 가우스 곡률 K |
| Chern 수 (정수) | ∫∫ K dA (연속) |
| |ψ⟩의 복소 위상 | ρ의 실수 확률 |

Fisher 계량은 양자 Fubini-Study 계량의 **고전 극한**이다:
```
g_μν^{Fisher} = lim_{ħ→0} g_μν^{Fubini-Study}
```

## Layer 1 의존성

Fisher 계량은 **볼츠만 분포** ρ ∝ exp(−V/T) 위에서 정의된다.
따라서:
- **FDT 필수**: σ² = 2γT/m → 올바른 ρ_eq 보장
- **T > 0 필수**: T=0이면 분포가 δ함수 → g_μν 발산
- **정규화 필수**: ∫ρ dx = 1 → 적분 영역 충분히 넓어야

## 인지적 해석

- **매개변수 = 환경 조건**: λ의 변화 = 외부 조건 변화
- **Fisher 계량 = 민감도**: g가 크면 "작은 환경 변화에도 분포가 크게 바뀜"
- **가우스 곡률 = 비선형 응답**: K ≠ 0이면 매개변수 변화의 효과가 비선형
- **측지선 = 최소 비용 경로**: 분포를 가장 적게 교란하는 매개변수 변화 경로

## 향후 확장 방향

1. **2D 물리 공간 + Berry phase**: 2D 비등방 시스템에서 진짜 Berry 위상
2. **정보 기하 → 최적 제어**: Fisher 계량 위의 측지선 = 최적 프로토콜
3. **Cramér-Rao bound**: g_μν 역행렬 = 매개변수 추정의 하한
4. **열역학 길이**: L_Fisher ∝ √(W_diss) (Layer 4 연결)
5. **양자 Fisher 정보**: ħ 보정 포함 (Parisi-Wu 확장 시)
